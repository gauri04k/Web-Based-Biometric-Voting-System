# Web-Based Biometric Voting System

A secure and efficient online voting platform developed using Python and Flask. This system leverages biometric authentication to verify voter identity, prevent duplicate voting, and ensure election transparency.
ytLink-https://www.youtube.com/watch?v=8mpMi6NXxqg

## 🚀 Features

- Secure Admin and Voter Authentication
- Biometric-Based Voter Verification
- Election Creation and Management
- Candidate Registration and Management
- Secure Vote Casting
- Real-Time Election Results
- Responsive User Interface
- Role-Based Access Control

## 🛠️ Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
- Bootstrap
- JavaScript

## ⚙️ Installation

```bash
git clone https://github.com/ShoyebChaudhari/Web-Based-Biometric-Voting-System.git
cd Web-Based-Biometric-Voting-System
pip install -r requirements.txt
python app.py
```

Open your browser and visit:

```bash
http://127.0.0.1:5000
```
